🧠 Why This Is Important

Without rate limiting:
Client
   │
   ▼
1000 Requests/Second
   │
   ▼
API Server
   │
   ▼
❌ Server Overloaded

With Redis Rate Limiting:
Client
   │
   ▼
Redis Counter
   │
   ├── Limit Not Reached ✔
   └── Limit Exceeded ❌

Benefits:
	•	✅ Prevents API abuse
	•	✅ Protects backend services
	•	✅ Works across multiple servers
	•	✅ Centralized request counting
